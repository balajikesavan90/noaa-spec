# Examples

This folder contains reviewer-friendly example artifacts.

- [docs/examples/noaa_demo](noaa_demo) contains lightweight HTML overview material.
- [docs/examples/stations](stations) contains three curated station-report examples kept in git.

The complete historical station-report set is archived locally under `data/archive/station_reports_full/`. This local archive is not required for reproduction.

Use [docs/REVIEWER_GUIDE.md](../REVIEWER_GUIDE.md) for the active reviewer path that connects these examples to installation, sample execution, and artifact interpretation.
