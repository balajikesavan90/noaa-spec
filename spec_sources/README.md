# Specification Source Artifacts

This directory keeps NOAA ISD documentation artifacts used for repository reference and provenance.

- `isd-format-document-parts/isd-format-document.pdf` is the preserved original NOAA PDF and the primary source-of-record for manual review.
- `isd-format-document-parts/isd-format-document.deterministic.md` is a derived, deterministic Markdown reference for searchable repository use.
- `isd-format-document-parts/part-*.md` files are derived reviewer-friendly navigation aids split by topic.

The derived Markdown files support inspection and navigation; they do not replace the original NOAA PDF as the preserved primary source artifact.
