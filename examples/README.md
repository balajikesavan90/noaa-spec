# Examples

Minimal runnable example for local first contact.

Run the installed-package example:

```bash
python3 examples/run_minimal_cleaning.py --out /tmp/noaa-spec-example.csv
```

Expected output file:

```text
/tmp/noaa-spec-example.csv
```

The script loads the bundled minimal NOAA sample, runs the cleaner, writes a deterministic CSV, and prints a preview of the result.
